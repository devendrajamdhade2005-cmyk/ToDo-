# Simple React Todo App

A clean and simple todo list application built with React.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks persist in localStorage
- Responsive design

## Getting Started

### Installation
```bash
npm install
```

### Run the app
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Build for production
```bash
npm run build
```
